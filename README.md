# Python Calculator App

A simple yet **powerful calculator** built in **Python** using **Tkinter**.  
Supports both **basic** and **advanced operations** with a friendly GUI.

---

## Features

- **Basic Operations:** +, -, ×, ÷  
- **Advanced Operations:** square, square root, exponentiation, percentage  
- **Keyboard Input:** type numbers and operations directly  
- **Clear & Backspace:** fix mistakes easily  
- **Light/Dark Mode:** toggle for better visibility
